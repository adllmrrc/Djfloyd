# DJ Floyd - Animation Son et Lumière

Site web pour DJ Floyd, prestataire d'animation, DJ, son, lumière pour événements (mariages, anniversaires, entreprises, etc.).

## Aperçu
- Page d'accueil moderne avec hero, équipe, contact, avis clients, devis, FAQ
- Formulaire de devis interactif (client-side validation)
- Design responsive, animations fluides
- Déployé sur Vercel (via vercel.json)

## Fichiers
- `index.html` - Page principale
- `legal.html` - Conditions générales / Mentions légales
- `script.js` - Interactions (menu mobile, form, scroll, animations)
- `styles.css` - Styles modernes (CSS variables, grid, flexbox, keyframes)
- `assets/` - Images (logo, galerie)
- `vercel.json` - Config déploiement
- `.gitignore` - Standard web

## Lancement local
Ouvrir `index.html` dans un navigateur.

## Déploiement Vercel
```
vercel --prod
```
(ou lien GitHub auto-deploy)
